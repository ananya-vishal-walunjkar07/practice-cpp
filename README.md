# 🚀 C++ Programs Repository

This repository contains a collection of C++ programs covering basic to intermediate concepts. It is useful for first-year engineering students and beginners to practice programming and understand core concepts.

---

## 📌 Features
- Beginner-friendly C++ programs
- Covers fundamental and OOP concepts
- Well-structured and easy to understand code
- Useful for exams and practice

---

## 🛠️ Technologies Used
- Language: C++
- Compiler: GCC / g++
- Concepts:
  - Variables & Data Types
  - Control Statements (if, loops)
  - Functions
  - Arrays & Strings
  - Pointers
  - Object-Oriented Programming
  - STL (if used)

---

## 📖 Programs Included
- Basic Input/Output
- Arithmetic Operations
- Conditional Statements
- Loops
- Functions
- Arrays
- Pointers
- Simple OOP Programs
